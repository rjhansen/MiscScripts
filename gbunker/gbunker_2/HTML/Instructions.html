Instructions for Use

Overview

Cardsharp is a tool for moving files from one directory (which we’ll call src) to another directory (which we’ll call dst) according to some specific rules. Once started, it will remain running until stopped (either by clicking on the stop button or by closing the window). You can drop files into src while the application is running and it won’t miss a beat.

Security considerations

This application will need access to your hard drive. If you click on “Help” and then “Report a Bug” it will attempt to open a web browser to point to GitHub. If you click on a hyperlink in this document it will try to open your default email client to send me a message. It does nothing else with the network, and may be run disconnected without any loss of important functionality.

So what does it copy?

Any file that has a four-digit 24-hour timestamp in the filename in the range 0000-2359. Cardsharp will interpret the timestamp as the time the file should be moved. At any given time in a given day, Cardsharp looks to find all files with a timestamp before the current moment. These files will be moved from src to dst.

What’s the center region displaying?

On the left you’ll see a list of all the timestamped files found in src. On the right you’ll see a list of all the timestamped files found in dst. This way you can at-a-glance ensure files are being moved appropriately.
